# Minimalist e-commerce API

1. Clone the project
2. `yarn install`
3. Clone the file `.env.template` and rename to `.env`
4. Change the environment variables
5. set up the data base.

```
docker-compose up -d
```

6. Get up the server:

```
npm run start:dev
yarn start:dev

```

7. Execute SEED

```
http://localhost:3000/api/seed
```
